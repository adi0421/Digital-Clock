# 🕐 Advanced Clock App

A real-time Analog + Digital Clock built with Python and Tkinter.

## Features
- Live digital clock with date and day
- Animated analog clock with hour, minute, and second hands
- Multi-timezone support (IST, UTC, EST, GMT, JST)
- Instant timezone switching
- Smooth minute hand movement

- Python 3.x

pip install pytz
python clock.py

## Tech Stack
- Python
- Tkinter (GUI)
- Pytz (Timezone)
- Math (Trigonometry for clock hands)
